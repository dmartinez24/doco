Doco is a simple, lightweight, and easy to use documentation generator for Markdown files. It
takes a directory of Markdown files and generates a static site with a table of contents.

Usage
    doco command [arguments]

Commands
    build     Generates the documentation site into /public/docs
    help      Prints the CLI help
    init      Creates a /docs directory with base files
    serve     Generates and serves the documentation site on localhost:8080
    version   Prints the CLI version
